Privacy Policy
Zero data collected — ever
Overview

We don't collect anything

DataCop is a browser extension that automatically dismisses cookie consent banners. It runs entirely inside your browser and has no backend, no servers, and no external connections of any kind.

This policy exists because Chrome Web Store requires one — not because there is any data to describe.

Data collection

What we do not collect

  • Browsing history or visited URLs
  • Page content or text
  • Cookies or local storage from websites
  • Personal or identifiable information
  • Crash reports or analytics
  • Usage statistics or telemetry
  • Any other data of any kind
Technical

How it works

DataCop injects a content script into web pages via Chrome's declarative content_scripts API. The script scans the DOM for cookie banner elements and hides them using CSS. That's it.

No network requests are made. No data leaves your device. The extension does not communicate with any server, including our own.

Permissions

Why we need host access

The extension requests access to all URLs (<all_urls>) because cookie banners appear on every website. Without broad host access, the extension cannot run on the sites you visit. This permission is used solely to inject the banner-blocking script — nothing else.
